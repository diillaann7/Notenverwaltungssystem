# Notenverwaltungssystem

Ein Spring Boot Backend-System zur Verwaltung von Schülernoten.  
Lehrer können Noten eintragen und die Noten einzelner Schüler abrufen. Das System nutzt eine relationale Datenbank und folgt einer sauberen Service-Controller-Repository-Architektur.

---

## **Projektbeschreibung**
Dieses Projekt demonstriert ein vollständig implementiertes Backend für die Notenverwaltung von Schülern.  
Es zeigt praxisnah, wie man eine **RESTful API** mit **Spring Boot**, **Datenbankintegration** und sauberer **Schichtenarchitektur** entwickelt.

**Features:**
- Noten für Schüler eintragen und verwalten
- Noten einzelner Schüler abrufen
- Nutzung von DTOs für sauberen Datentransfer
- Service-Schicht für Geschäftslogik
- Repository-Schicht für Datenbankzugriff
- REST-Controller für API-Endpunkte

---

## **Technologien**
- **Java 17** – Programmiersprache
- **Spring Boot** – Framework für Backend-Entwicklung
- **Spring Data JPA** – Einfacher Datenbankzugriff über Repositories
- **H2 / MySQL / PostgreSQL** – Relationale Datenbank (H2 für Demo, MySQL/PostgreSQL für Produktion)
- **DTO Pattern** – Sauberer Datentransfer zwischen Schichten
- **Maven** – Build- und Dependency-Management
- **Spring Security (optional)** – Benutzerrollen (Lehrer/Schüler) für Zugriffskontrolle
